Programa de Referidos Bluemeds

Términos y Condiciones

1. El programa de referidos aplica únicamente si la persona que refiere consume un monto mensual igual o mayor a Q.300 así como también la persona referida debe ingresar con un monto igual o mayor a Q.300 Esto aplicará para la siguiente entrega.

2. Cada persona suscrita a Bluemeds tiene un código único para poder compartir con sus familiares y amigos. Si compartes otro código que no es el tuyo, no se aplicará ningún descuento.

3. Para que el descuento se aplique, a la nueva persona suscrita se le debe realizar el primer cobro de forma exitosa.

4. Si el monto de la entrega es menor al monto del descuento acumulado (según la cantidad de personas referidas y suscritas exitosamente), se le cobrará Q1.00 al suscriptor y la diferencia no se acreditará a una entrega posterior.

5. El código puede usarse múltiples veces. Se harán Q75 de descuento por cada suscripción exitosa que la persona refiera.
